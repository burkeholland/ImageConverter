Installation

1

Download

Download the latest release ZIP file from the button above or the releases page.

2

Extract

Extract the ZIP file to any folder on your computer.

3

Install

Right-click Install.bat and select Run as administrator. This adds the app to your Start Menu and enables the right-click context menu.

4

Convert!

Right-click any image file and select Convert Image... to start converting.
